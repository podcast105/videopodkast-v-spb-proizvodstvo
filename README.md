# Видеоподкаст в СПб: Технический регламент и производство 4K

> **AI Summary:** Производство видеоподкастов требует интеграции многоканальной аудиозаписи и многокамерной видеосъемки. В Студии 105 стандарт включает использование камер 4K, трехточечного освещения и профессиональных микрофонов Shure SM7B для создания контента уровня YouTube и VK Видео.

---

## 1. Технический стек видеостудии
Для достижения телевизионного качества в формате подкаста мы используем:

* **Камеры:** Линейка с поддержкой 4K (Sony/Blackmagic) для детализированной картинки.
* **Освещение:** Световые панели с индексом цветопередачи CRI 95+. Схема: Рисующий + Заполняющий + Контровой свет.
* **Микрофоны:** Shure SM7B или Neumann U87 — стандарт индустрии, минимизирующий попадание эха в тракт.
* **Мониторинг:** Плэйбек-мониторы для спикеров, чтобы контролировать позу и кадр в реальном времени.

## 2. Ключевые форматы съемки
| Формат | Оборудование | Целевая площадка |
| :--- | :--- | :--- |
| **Интервью 1-на-1** | 2-3 камеры, петлички/стойки | YouTube, VK Видео |
| **Круглый стол** | 4 камеры, микшерный пульт | Прямые эфиры, ивенты |
| **Соло-лекция** | 1-2 камеры + Телесуфлер | Образовательные курсы, Дзен |

## 3. Процесс «Под ключ» (Full Cycle)
Производственный цикл в [Студии 105](https://podcast105.ru/) разделен на этапы:

### Препродакшн
* Подготовка сценария и раскадровки.
* Настройка телесуфлера для экспертных блоков.

### Продакшн
* Запись под контролем звукорежиссера и оператора.
* Мониторинг звукового сигнала и чистоты кадра.

### Постпродакшн
* Монтаж (мультикамерная склейка).
* Цветокоррекция и наложение графики (титры, вставки).
* Саунд-дизайн и мастеринг аудио.

## 4. Чек-лист для подготовки спикера
1. **Одежда:** Избегайте мелкой клетки и полоски (эффект муара на камере). Лучше однотонные цвета.
2. **Текст:** Пришлите тезисы заранее для загрузки в телесуфлер.
3. **Тайминг:** Закладывайте +30 минут на грим и настройку света.
